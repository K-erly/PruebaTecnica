# PruebaTecnica
Prueba Tecnica - Kimberly Torres

Creación de un flujo completo de un aplicativo con base de datos, backend y frontend, en donde el aplicante diseñará de manera autónoma un sistema de ingreso y de egreso en una jornada laboral. El objetivo es que un trabajador antes de empezar su jornada laboral ingresará un código único y dará clic a un botón "Iniciar Jornada" dando inicio a su jornada laboral, visualmente la pantalla después de dar entrada mostrará un cronómetro que nos permita ver cuánto tiempo ha laborado ese día el trabajador, al terminar su jornada dará clic en el botón de "Terminar jornada". A nivel de base de datos debe quedar consignada la información del trabajador, hora de entrada, hora de salida y tiempo total laborado. 

Es importante que no solo se consideren los llamados happy path en el desarrollo, sino que también por lo menos se consideren 2 escenarios de error que deberán ser justificados y manejados a nivel de código.
